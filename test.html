jqSprites

A simple jQuery plugin that let you easily use and manage (rollovers, animations) sprites into <img> elements.

Fork on Github

  1. Use the script 
    $(function() {
    
    $('.selector').sprites({
        'defaultWidth' : 40,
        'defaultHeight' : 60
    });
    
});

  2. Simple sprite with X and Y. It takes defaultWidth and defaultHeight as size.

    <img src="sprite.png?x=20&y=0">

  3. Simple sprite with X, Y and a class

    <img src="sprite.png?x=200&y=0" width="40" height="60" class="pics">

  4. Rollover with additional hoverX and hoverY

    <img src="sprite.png?x=140&y=0&hx=80&hy=0" width="40" height="60" class="pics">

  5. Linear Animation

    <img src="sprite.png?x=20&y=0&animate=true&offset=40&end=200&fps=10&loop=true" width="40" height="60" class="pics">
